# Vue 3 App

This is a Vue 3 application. Below are instructions on how to start the app using NPM or Docker.

## How to Start the App

### If You Use NPM

1. Install dependencies:
    ```sh
    npm install
    ```

2. Build the project:
    ```sh
    npm run build
    ```

3. Preview the built project:
    ```sh
    npm run preview
    ```

4. Open http://localhost:3000/ url in your browser

### If You Use Docker

1. Start the app using Docker Compose:
    ```sh
    docker-compose up
    ```

4. Open http://localhost:3000/ url in your browser

## Additional Information

- Make sure you have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed if you choose to use NPM.
- Make sure you have [Docker](https://www.docker.com/) and [Docker Compose](https://docs.docker.com/compose/) installed if you choose to use Docker.
